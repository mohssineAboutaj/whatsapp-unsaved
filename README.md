## About

A very basic [ionc/vue](https://ionicframework.com/docs/vue/overview) app that let you chating with any whatsapp user without save his/her phone number in your device

## Demo

## Project setup

### Installation

```sh
yarn install
```

### Development

```sh

```

### Production

```sh

```
